# vscode

## code コマンドライン (mac のみ)

`Shift + Command + P` で `Shell Command` と入力

### 拡張機能一覧更新

```
code --list-extensions > extensions.txt    
```

### インストール

```
./install.sh
```

### アンインストール

```
./uninstall.sh
```

## 設定ファイル

```
cp ~/Library/Application\ Support/Code/User/settings.json ./settings.json  
```
